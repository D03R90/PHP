$a = 5;

$b = '05';

var_dump ( $a == $b ); // Почему true?

(Одинаковый тип переменных, строка b была преобразована к числу)



var_dump (( int ) '012345' ); // Почему 12345?

(Integer приводит к целому числу)



var_dump (( float ) 123.0 === ( int ) 123.0 ); // Почему false?

(Разный тип переменных)



var_dump (( int ) 0 === ( int ) 'hello, world' ); // Почему true?

(Неявное преобразование типов, чтобы сравнить два разных типа, нужно один привести к другому. )
